# Glossary

SEO / backlink terms used across the knowledge base. Keep definitions short and source-neutral;
if experts define a term differently, note both in the relevant strategy's "Conflicting views".

| Term | Definition |
|---|---|
| Backlink | A link from one website to another; a ranking signal. |
| DR (Domain Rating) | Ahrefs' 0–100 score of a domain's backlink-profile strength. |
| DA (Domain Authority) | Moz's 0–100 equivalent score. |
| Anchor text | The clickable text of a link; signals relevance (exact-match, partial, branded, naked URL, generic). |
| Link velocity | The rate at which a site gains/loses backlinks over time. |
| Dofollow / Nofollow | Whether a link passes ranking signal (`rel="nofollow"` asks engines not to). |
| Referring domain | A unique domain linking to you (more valuable than raw link count). |
| Link equity / "link juice" | The ranking value passed through a link. |
| PBN | Private Blog Network — a set of owned sites built to link to a money site (black-hat). |
| HARO / Connectively | Service connecting journalists with sources; a digital-PR link tactic. |
| Linkable asset | Content created specifically to attract links (stats, tools, guides). |
| Link insert / niche edit | Adding a link into an existing published article. |
| Tiered link building | Building links to your backlinks to boost their power. |
| Toxic / spam link | A low-quality link that may trigger penalties. |
| Link gap analysis | Comparing competitors' backlinks to find link opportunities you lack. |
| Reverse silo | Acquiring external links to linkable assets that internally link to commercial/money pages, spreading authority and cutting penalty risk. (source 0001) |
| Linkable asset / link bait | Content engineered to attract backlinks (calculators, tools, infographics, rankings, data studies, glossaries). (source 0001) |
| Linking root domains | Count of unique referring domains to a page — Gotch's #1 signal that an idea is linkable. (source 0001) |
| User signals | Audience engagement (e.g. Reddit upvotes/comments) used as a secondary link-bait validation signal. (source 0001) |
| Relevancy pyramid | Gotch's framework for bucketing link prospects by relevance and working top-down. (source 0001) |
| Dream 100 | A curated list of the 100 highest-value target sites in a niche for relationship-led link building. (source 0001) |
| Referral-traffic principle | Prioritize link prospects by whether they'd send relevant, conversion-likely referral traffic. (source 0001) |
| WIIFM ("what's in it for me") | Outreach must lead with the recipient's benefit; the value proposition is the deciding factor. (source 0001) |
| Expert roundup | An article compiling multiple experts' input; a placement target for figurehead link building. (source 0001) |

| Link intent | A keyword's propensity to attract passive backlinks when ranked for — ~20% of keyword targets should have high link intent. (source 0002) |
| Anchor text over-optimization | Using the same exact-match commercial keywords as anchor text across many links — a manipulative pattern Google penalizes. (source 0002) |
| Content resurrection | Finding expired-domain pages that still have quality backlinks, creating a better replacement, and outreaching to sites linking to the dead page. (source 0002) |
| Content replacement | Finding outdated content on live trusted sites, creating a newer version, and pitching it as the replacement. (source 0002) |
| Link Intersect / Backlink Gap | Finding sites that link to your competitors but not to you — a shortcut to pre-qualified link opportunities. (source 0002) |
| Like-to-Link method | Studying a trusted site's outbound links to discover what content types that site is willing to cite — then creating similar content. (source 0002) |
| Answer bait | Informational pages that directly answer a specific question ("Can I pour bleach into the sink?") — earn links because writers cite them as references. (source 0002) |
| Marketer bait | Content targeting marketing, advertising, and SEO bloggers — even when your primary niche is unrelated — to earn links from highly link-generous domains. (source 0002) |
| Topical Trust Flow | Majestic's metric scoring a domain's relevance to a specific topic category; used in SpamZilla to filter expired domains by niche. (source 0002) |
| Disavow | Telling Google to ignore specific backlinks when evaluating your site — via the Search Console Disavow tool. High risk; Gotch recommends it in only 0.1% of cases. (source 0002) |
| NAP-W | Name, Address, Phone, Website — the core business details that should match across all listings (and your Google Business Profile) so Google trusts they refer to the same business. (source 0002) |
| Google Business Profile | Google's free business listing (formerly Google My Business) that powers Maps/local results; the reference point for consistent NAP-W. (source 0002) |
| Link roundup | A recurring (often weekly) post that curates and links to the best new content in a niche; a placement target. (source 0002) |
| Reverse expert roundup | Pitching to be *added* to an already-published expert roundup, rather than running your own. (source 0002) |
| Resource page | A page maintained specifically to link out to helpful resources on a topic; a placement target. (source 0002) |
| 301 redirect | A permanent "this page moved" instruction that passes most of the old URL's link value to the new destination. (source 0002) |
| Merger technique | Acquiring a live site/tool and 301-redirecting its pages into yours to inherit its backlinks (expired domains now discouraged). (source 0002) |
| Authority Score (AS) | Semrush's 0–100 measure of a domain's overall quality/authority (its DR/DA equivalent). (source 0002) |
| Anchor text cycling | Rotating through anchor types (branded, generic, naked, partial, exact, etc.) across links to keep the overall anchor profile natural. (source 0002) |
| Passive link magnet | An asset so useful or structurally required (free micro-tool, exhaustive docs, social profile, privacy/ad policy) that sites link to it at scale without outreach. (source 0002) |
| Email warm-up | Gradually ramping send volume from a new email address (ideally aged + company-branded) to build deliverability before scaling outreach. (source 0002) |
| Followed link | A link with no `rel="nofollow"`, so search engines count it as a ranking vote. (source 0002) |
