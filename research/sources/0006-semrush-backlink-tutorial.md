## Summary
A Semrush product tutorial on finding **better** (not just more) backlinks three ways: (1) **broken
backlink building** using Domain Overview → Backlink Analytics; (2) the Semrush **Link Building Tool**
(find relevant prospects + reach out from one place); and (3) the **Backlink Gap** tool (sites linking to
competitors but not you). Most of this is already in the KB (Backlink Gap → `link-prospecting-prioritization`;
paid-link penalties → `buy-backlinks`/`toxic-backlink-audit`; anchors → `anchor-text-optimization`). The
net-new bits captured here: the **generic live-site broken-link workflow** (Indexed Pages → Broken Pages)
plus the **Ctrl+F anchor trick**, and the **Link Building Tool** prospect→outreach workflow.

  Indexed Pages → **Broken Pages** (sorted by referring domains) → infer the dead page's content → match

## Notable quotes / unique insights
- "Quality over quantity" — and **referring domains usually matter more than raw backlink count**.
- Broken link building is a **win-win-win**: better UX for the reader, healthier SEO for the owner, a link
  for you — and it's low-effort for the owner to swap a dead link for a relevant live one.
- **Ctrl+F (Cmd+F) the anchor text** on a linking page to jump straight to the broken link.
- Don't buy cheap backlinks — paid follow links are link spam and can get a site **penalized / removed**;
  paid links must be `nofollow`/`sponsored`. (Already in [[strategies/buy-backlinks]].)
- Backlink Gap drill-down can reveal a competitor was **linked then had the link removed** — an opening to
  pitch your page as the replacement.

## Raw notes
- **Domain Overview** (Competitor Research): enter a competitor (demo used chewy.com) → Authority Score,
  organic + paid traffic, backlinks, referring domains, country breakdown, organic/paid toggle.
- **Backlink Analytics report:** backlink types (text / image / form / frame), link attributes
  (follow / nofollow / sponsored / ugc), referring domains vs backlinks, per-link Authority Score, anchor +
  target URL. (Educational; intentionally not added to the KB glossary per scope.)
- **Broken link building:** Link Building → Backlink Analytics → competitor → **Indexed Pages** → check
  **Broken Pages** → list sorted by # linking domains → infer content from title/URL → skip non-mappable
  (e.g. login page) → match an existing page or create one → open a linking page and **Ctrl+F the anchor
  text** to locate the broken link → find contact (email/social/contact form) → outreach.
- **Link Building Tool:** enter your domain + target keywords (delete the bad auto-keywords, add real ones,
  e.g. "organic dog food") → it analyzes and returns **prospects** (sites linking to competitors and/or
  ranking for the keywords) with domain, snippet, URL type, **Authority Score**, and a **fit rating** →
  open each to vet → add good ones to **In Progress** (trash the rest) → pick an **Outreach strategy**
  (e.g. directory/catalog, guest post) which pre-fills an **email template** → tool **auto-finds contact
  emails** + provides placeholders (domain/URL) → edit subject/body → **save as template** → **send + track**
  from within the tool.
- **Backlink Gap** (Competitive Research): your domain + competitors → **Find prospects** → sorted by
  **Best** (link to all competitors, not you) → expand a row to see which competitor page links out → spot
  comparison pages, list posts, directories, and removed-link opportunities.
- Repeated framing: backlinks take **time / a consistent process**, not one-and-done; promotes Semrush's
  free academy SEO course.
