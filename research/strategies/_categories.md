# Strategy Categories (controlled vocabulary)

Use exactly one of these slugs in a strategy file's `category:` frontmatter so tagging stays
consistent across the whole knowledge base. Add a new category here first if a strategy genuinely
doesn't fit — don't invent ad-hoc values inline.

| Category | Slug | What belongs here |
|---|---|---|
| Outreach | `outreach` | Manual relationship/email-driven link building (guest posts, link inserts, broken-link outreach). |
| Content-driven | `content-driven` | Earning links by creating linkable assets (stats pages, guides, tools, original research). |
| Digital PR | `digital-pr` | Journalist/media coverage, HARO/Connectively, newsjacking, data studies pitched to press. |
| Technical | `technical` | Internal linking, link reclamation via redirects, fixing lost links, schema. |
| Link reclamation | `link-reclamation` | Recovering unlinked mentions, broken backlinks, stolen-image links. |
| Directories & citations | `directories-citations` | Niche directories, local citations, business listings. |
| Community | `community` | Forums, Q&A sites, Reddit, Slack/Discord, social profiles. |
| Paid | `paid` | Sponsored posts, paid placements (flag risk; often violates guidelines). |
| Black-hat | `black-hat` | PBNs, link farms, comment spam — documented for awareness only, always flagged. |
