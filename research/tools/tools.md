## ahrefs
- **Name:** Ahrefs
- **Category:** backlink-analysis
- **What it does:** Backlink index, Domain Rating (DR), referring domains, top pages — used to gauge link authority and reverse-engineer competitors' most-linked pages. Also: Site Explorer → Outgoing Links → Linked Domains (confirm a site links out); Backlinks search filters (find roundups/resource pages/guest posts/donor pages by referring-page keyword); Keyword Explorer → Link Intersect (niche-edit prospecting).
- **Used in strategies:** [[strategies/reverse-silo]], [[strategies/linkable-asset-creation]], [[strategies/link-prospecting-prioritization]], [[strategies/get-interviewed]], [[strategies/niche-business-associations]], [[strategies/donation-link-building]], [[strategies/link-roundups]], [[strategies/reverse-expert-roundups]], [[strategies/list-post-link-building]], [[strategies/resource-page-link-building]], [[strategies/merger-technique]], [[strategies/guest-posting]], [[strategies/niche-edits]], [[strategies/oprah-technique-podcast]]
- **Pricing:** paid (subscription)
- **Recommended by:** Nathan Gotch (source 0001, source 0002)

## semrush
- **Name:** Semrush
- **Category:** backlink-analysis
- **What it does:** Broad industry/keyword search + Backlink Analytics → Index Pages (sortable by referring domains) to find competitors' most-linked page types. Also: **Domain Overview** (competitor bird's-eye — Authority Score, organic/paid traffic, backlinks, referring domains); Backlink Analytics → **Indexed Pages → Broken Pages** (broken-link prospecting); the **Link Building Tool** (domain + keywords → prospects with AS + fit rating, auto contact-finding, templated in-tool outreach); Backlink Audit (toxicity scoring); Backlink Analytics → Outbound Domains (Like-to-Link method); Backlink Gap (Link Intersect); Keyword Overview (Dream 100 discovery); Bulk Analysis (bulk URL authority scoring).
- **Used in strategies:** [[strategies/reverse-silo]], [[strategies/linkable-asset-creation]], [[strategies/link-prospecting-prioritization]], [[strategies/toxic-backlink-audit]], [[strategies/link-bait-generation-methods]], [[strategies/content-resurrection]], [[strategies/jobs-page-link-building]], [[strategies/get-interviewed]], [[strategies/donation-link-building]], [[strategies/resource-page-link-building]], [[strategies/merger-technique]], [[strategies/buy-backlinks]], [[strategies/outreach-execution]]
- **Pricing:** paid (subscription)
- **Recommended by:** Nathan Gotch (source 0001, source 0002); Semrush (source 0006)

## featured

## backlinker-ai
- **Name:** BacklinkerAI (backlinker.ai)
- **Category:** automation
- **What it does:** Automates the journalist-request ("HARO-style") play end-to-end. You set up one **persona** (name, LinkedIn, company, job title, website, bio); the tool then uses **Featured** in the background to auto-write and submit expert pitches as that persona to journalists seeking sources. Accepted pitches become published articles with an editorial backlink. Dashboard tracks pitches → selected/in-review → published. Solo and Agency modes. Tagline: "Get 65% more SEO traffic with outreach backlinks on autopilot"; advertises ~3 guaranteed links.
- **Used in strategies:** [[strategies/haro-journalist-requests]]
- **Pricing:** ~$300/mo (prorated mid-cycle); affiliate code `letsuncover` advertised at 50% off (~$150/mo) in source 0003.
- **Recommended by:** Usman Tariq (source 0003) — **vendor walkthrough with an affiliate code and self-reported metrics; vet placements/quality independently before client use.**
